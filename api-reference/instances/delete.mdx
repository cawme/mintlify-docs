`DELETE /v1/instances/:instanceId`

Realiza o soft-delete de uma instância, definindo seu status como `DELETED`. O registro é mantido no banco de dados, mas não é mais retornado nas operações de listagem ou busca.

## Requisição

### Cabeçalhos

| Cabeçalho | Valor |
|---|---|
| `Authorization` | `Bearer <token>` |

### Parâmetros de caminho

| Parâmetro | Tipo | Descrição |
|---|---|---|
| `instanceId` | `string` | O ID da instância a ser deletada |

## Resposta `204`

Sem conteúdo. A instância foi deletada.

## Erros

| Código | Descrição |
|---|---|
| `401` | Token ausente ou inválido |
| `404` | Instância não encontrada, já deletada ou não pertence ao usuário |

## Exemplo

```bash
curl -X DELETE https://cawme.com/api/v1/instances/inst_abc123 \
  -H "Authorization: Bearer <token>"
```

<Warning>
  A exclusão é permanente — instâncias deletadas não podem ser restauradas.
</Warning>
