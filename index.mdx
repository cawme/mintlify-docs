## O que é o Cawme?

O Cawme permite que empresas realizem chamadas de voz no WhatsApp de forma programática para seus clientes. Seja para enviar uma mensagem TTS fixa, reproduzir um arquivo de áudio pré-gravado, transmitir áudio em tempo real ou implantar um agente de IA para conversas dinâmicas — o Cawme cuida de tudo por meio de uma API REST simples.

## Conceitos principais

- **Instância** — Um número de WhatsApp vinculado ao Cawme. Cada instância possui sua própria fila de chamadas e avança pelos status: `CREATED`, `CONFIGURED`, `ACTIVE` ou `DELETED`.
- **Chamada** — Uma chamada de saída iniciada por uma instância. As chamadas passam por: `QUEUED → RINGING → IN_PROGRESS → COMPLETED / FAILED`.
- **Tipos de chamada** — Quatro modos: [`FIXED_TTS`](/guides/call-types#fixed-tts), [`FIXED_AUDIO`](/guides/call-types#fixed-audio), [`DYNAMIC_STREAM`](/guides/call-types#dynamic-stream) e [`DYNAMIC_AGENT`](/guides/call-types#dynamic-agent).

## URL base

```
https://cawme.com/api/v1
```

## Autenticação

O Cawme utiliza o **Clerk** para autenticação. Inclua um Bearer token em todas as requisições:

```http
Authorization: Bearer <seu-token>
```

Consulte o [guia de Autenticação](/guides/authentication) para mais detalhes.
